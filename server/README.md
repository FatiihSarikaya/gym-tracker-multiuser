# GymTracker Node.js Backend

API base URL: `http://localhost:5000/api`

Routes

- Members: `/api/Members`
- Attendance: `/api/Attendance`
- Payments: `/api/Payments`
- Lessons: `/api/Lessons`
- LessonAttendances: `/api/LessonAttendances`

Run locally

```bash
cd server
npm install
npm run dev
```

Data is persisted in `server/data/db.json` using lowdb.
